This pack contains the following files:

  toledo_javascript_chess_3.html
    Toledo Javascript Chess, using only Unicode
    graphics for displaying pieces. 2299 bytes.

  toledo_javascript_chess_1.html
    Toledo Javascript Chess, using only letters for
    displaying pieces. 2159 bytes.

  toledo_javascript_chess_2.html
    Toledo Javascript Chess, using graphics for
    displaying pieces. 3130 bytes.

  0.gif     Empty square
  1.gif     Black pawn
  2.gif     Black king
  3.gif     Black knight
  4.gif     Black bishop
  5.gif     Black rook
  6.gif     Black queen
  9.gif     White pawn
  10.gif    White king
  11.gif    White knight
  12.gif    White bishop
  13.gif    White rook
  14.gif    White queen
    The GIF pieces can be anything up to 40x40 pixels
    (currently 8x8 for saving space)

  tiny_chess_1.html
    Tiny Chess (1023 bytes of Javascript!)

    Program sent to the JS1K contest, won 2nd place!
    Check it at:
      http://js1k.com/
      http://js1k.com/demo/750/

    Play chess against computer with queen-only promotion
    and without castling or en passant.

    Click origin square and then target square to move
    piece, it will validate moves.

    Uses Unicode graphics for the chess pieces.

  tiny_chess_2.html
    Same program but with castling and en passant, only
    uses 1251 bytes of Javascript!.

  readme.txt
    This file.

Play level of all programs is fixed at 3-ply depth, you can
increase it editing the source code (only if you have a very
fast browser).

Visit http://www.nanochess.org for more information and other
interesting bits.
